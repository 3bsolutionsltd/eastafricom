# East Africom Dynamic Website System

Professional coffee and cocoa export website with dynamic content management.

## Features
-  Dynamic database-driven content
-  Admin dashboard with CRUD operations
-  28 authentic Uganda coffee varieties
-  Mobile-responsive design
-  Automated CI/CD deployment
-  Environment-aware configuration

## Environments

### Local Development (WAMP)
- **URL:** http://localhost/eastafricom/
- **Admin:** http://localhost/eastafricom/admin/
- **Database:** eastafricom_cms

### Production (Hostinger)
- **URL:** https://live.eastafricom.com
- **Admin:** https://live.eastafricom.com/admin/
- **Database:** u266222025_eastafricom

## Development Workflow
1. Make changes locally on WAMP server
2. Test functionality thoroughly
3. Commit changes: `git add . && git commit -m "Description"`
4. Push to GitHub: `git push origin main`
5. Automatic deployment to live server

## Project Structure
- `/admin/` - Admin dashboard
- `/backend/` - PHP API and database
- `/css/` - Stylesheets
- `/js/` - JavaScript modules
- `/images/` - Media assets
- `/pages/` - Static pages

## Quick Start
1. Set up local WAMP environment
2. Import database: eastafricom_cms
3. Clone this repository to C:\wamp64\www\eastafricom
4. Access: http://localhost/eastafricom/
